# 国家大学生就业服务平台使用指南

`更新 26-02-01` \| `发布 26-02-01`

听说假期中可以关注该服务平台，找到招聘信息，一起看看怎么使用吧。网站链接：[国家大学生就业服务平台↗]

## 网站简介

（原文：[网站简介↗]）

国家大学生就业服务平台是由教育部主管、教育部学生服务与素质发展中心运营的服务于高校毕业生及用人单位的公共就业服务平台。

- **精准推送职位**。根据往年招聘情况精准推送至毕业生，精准共享至各省、各校就业网，实现高精准、广覆盖。
- **线上线下结合**。采取手机版、二维码等移动互联网模式实现线上线下服务相结合。
- **大型网络招聘会**。定期举办全国、各省市联合招聘会、重点领域招聘会，构建人才招聘零距离通道。
- **生源信息查询**。分学历、专业查询全国800多万高校毕业生生源分布。
- **大数据支撑**。每年800多万就业数据、800多万生源数据、学信网1000万新增用户以及3500万在校生数据库、1.7亿高等教育学历信息库，为用人单位精准、高效招聘提供数据与用户支撑。
- **学籍学历验证**。求职者学籍学历信息经过验证，单位也可自主查询验证。
- **重点领域带动**（可查看 [重点领域](#重点领域)）。推动高校毕业生到重点领域单位就业，实现高校毕业生更高质量更充分就业。
- **专属客户和推广服务**。专属客服，提供最优解决方案；专属品牌及知名度推广服务。

### 投递简历

如想投递简历，该平台要求先注册登录。可以注册成功的。然后需要输入这个那个，比如个人信息、教育背景、求职期望、实习经历，等等。**这些信息，可能都可以不填的。**

**建议：要上传附件简历。** 附件简历就是已经写好的、pdf版本的简历。因为在简历投递时，该平台提示：

```markdown
简历投递确认

您正在投递 xxxx（用人单位） 的 xxxx（求助岗位）
您已上传附件简历，投递时会发给用人单位
```

### 重点领域

更多信息，可查看 [重点领域↗] 页面。此处摘录 **重点领域推荐单位申请须知**（[原文链接↗](https://www.ncss.cn/student/keyunits/intropart.html)），供了解该平台的重点领域是指什么。一句话：是推荐产生的。摘录部分信息如下：

为鼓励和引导高校毕业生到国家重点领域就业，教育部学生服务与素质发展中心推出“重领域就业信息服务平台”，促进“双一流”高校与重点领域单位校企对接，现开展重点领域位推荐工作。

国家大学生就业服务平台及省级、校级就业网将为入选的重点领域推荐单位共享和精准送职位信息，并提供重点领域校企对接、网络招聘、重点推介宣传、生源信息查询等专免费服务。具体申请流程如下：

1. 申请单位填写《重点领域推荐单位申报表》（下载表格）；
2. 申请单位请推荐部门（省级就业指导中心或高校就业工作部门）签署推荐意见；
3. 将填写完毕的《重点领域推荐单位申报表》扫描发送至我中心联系人电子邮（wangym#chsi.com.cn 将#替换成@）。

申请推荐成功后，单位信息将纳入“重点领域就业信息服务平台”，获得“重点领域”专属识，并短信通知申请成功。
