# 简历优化建议

## T20F-2601

`更新：26-01-29` \| `发布：26-01-29`

W同学好！

针对 26-01-28 版求职简历，有如下建议，供参考。

### 基本信息

- 出生年月，可不写。
- 团员，可不写。
- 可参考 [基本信息↗]， 看看还有哪些有助于获取 offer 的信息，能呈现在简历中。

### 教育背景

- 绩点。似乎不高，建议不写。如果要写，建议增加专业排名信息，比如：GPA 3.xx（专业排名前xx%）。
- 可参考 [教育背景↗]，看看还有哪些有助于获取 offer 的信息，能呈现在简历中。

### 项目经历

- 描述方式类似记流水账，没有呈现出“我很厉害！我很适合该岗位！”
- 请参考 [项目经历↗]，重新描述该段落。
- 如果篇幅受限，则选取和所投岗位相关的几个项目，展开描述即可。不大相关的项目，就删除吧。

### 社会实践

- 2个社会实践的描述，似乎和 [项目经历] 段落很重复。建议都删除吧。

### 技能证书

- 建议 [技能证书] 改成 [专业技能]。
- 参考所投岗位的专业技能的要求，列出你的能力。要什么写什么，而不是有什么写什么。
- 不要把英语能力放第一个。
- 不要只写“熟悉C、C++语言”这几个字，要略展开描述，写满一行。
- 不要把“MySql”（应该是MySQL）和 “html”等写在一行，且这些也不是什么“工具”。
- 请参考 [专业技能↗]，重新描述该段落。

### 自我评价

- 参考所投岗位对软能力的要求，对现有描述是适当增删。要什么写什么，而不是有什么写什么。

### 其他建议

- 缺 [校园经历] 段落，建议补充。
- 要正确使用中文的标点符号。
- 要正确使用英文的大小写。
- 简历发送给用人单位时，建议使用 pdf 格式或高清截图，不是用 Word 格式。
- 有空可看看 [求职简历建议汇总↗] 全文，希望对你所有启发， 将更多精彩内容呈现在简历上。

预祝早日找到心仪岗位！🎉🎉🎉
